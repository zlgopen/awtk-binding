详细描述

spinbox控件。

一个特殊的数值编辑器，将edit_t和button_t进行组合，方便编辑数值。

点击向上的按钮将数值增加一个step，点击向下的按钮将数值减小一个step。 step的值可以通过step属性进行设置。

spin_box_t是edit\_t的子类控件，edit_t的函数均适用于spin_box_t控件。

在xml中使用"spin_box"标签创建spinbox控件。如：

*<spin_box w="70%" input_type="int" min="-100" max="100" step="5">

更多用法请参考：spin_box.xml
