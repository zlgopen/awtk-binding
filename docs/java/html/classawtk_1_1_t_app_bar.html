详细描述

app_bar控件。

一个简单的容器控件，一般在窗口的顶部，用于显示本窗口的状态和信息。

它本身不提供布局功能，仅提供具有语义的标签，让xml更具有可读性。 子控件的布局可用layout_children属性指定。 请参考布局参数。

app_bar_t是widget\_t的子类控件，widget_t的函数均适用于app_bar_t控件。

在xml中使用"app\_bar"标签创建app_bar。如：

*<app_bar x="0" y="0" w="100%" h="30"
*<label x="0" y="0" w="100%" h="100%" text="Basic Controls" />
*</app_bar>

*<normal bg_color="#f0f0f0" />
*</style>
