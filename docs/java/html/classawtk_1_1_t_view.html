详细描述

一个通用的容器控件。

它本身不提供布局功能，仅提供具有语义的标签，让xml更具有可读性。 子控件的布局可用layout_children属性指定。 请参考布局参数。

view_t是widget\_t的子类控件，widget_t的函数均适用于view_t控件。

在xml中使用"view"标签创建view。如：

*<view x="0" y="0" w="100%" h="100%" children_layout="default(c=2,r=2,m=5,s=5)">
*</view>

可用通过style来设置控件的显示风格，如背景颜色等。如：

*<style name="default" border_color="#a0a0a0">
*<normal bg_color="#f0f0f0" />
*</style>

缺省获得焦点的子控件(可用控件名或类型)。

view作为pages/slideview的直接子控件才需要设置。 正常情况下，一个窗口只能指定一个初始焦点。 但是对于pages/slideview来说，可能希望每一个页面都有一个初始焦点，此时可用default_focused_child来指定。
