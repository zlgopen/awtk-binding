详细描述

定时器。

主要目的是方便以拖拽的方式创建定时器，并用AWBlock编写简单的事件处理程序。

在xml中使用"timer"标签创建控件。如：

*<!-- ui -->
*<timer x="c" y="50" w="100" h="100" duration="1000"/>

可用通过style来设置控件的显示风格，如字体的大小和颜色等等。如：

本控件默认不可见，无需指定style。

*<!-- style -->
*<timer>
*<style name="default" font_size="32">
*<normal text_color="black" />
*</style>
*</timer>
