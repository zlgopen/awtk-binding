详细描述

用于串口通信的控件 在xml中使用"serial"标签创建控件。如：

*<!-- ui -->
*<serial device="COM1" baudrate="115200"/>

可用通过style来设置控件的显示风格，如字体的大小和颜色等等。如：

本控件默认不可见，无需指定style。

*<!-- style -->
*<serial>
*<style name="default" font_size="32">
*<normal text_color="black" />
*</style>
*</serial>
