◆ CHANGED

awtk.TBitmapFlag.CHANGED =(BITMAP_FLAG_CHANGED())

如果是MUTABLE的图片，更新时需要设置此标志，底层可能会做特殊处理，比如更新图片到GPU。

◆ GPU_FBO_TEXTURE

◆ IMMUTABLE

◆ LCD_ORIENTATION

awtk.TBitmapFlag.LCD_ORIENTATION =(BITMAP_FLAG_LCD_ORIENTATION())

位图数据已经处理了 lcd 旋转，同时说明 bitmap 的宽高和真实数据的宽高可能不一致
