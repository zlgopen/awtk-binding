详细描述

Button Group控件。一个简单的容器控件，用于容纳一组按钮控件。

它本身不提供布局功能，仅提供具有语义的标签，让xml更具有可读性。 子控件的布局可用layout_children属性指定。 请参考布局参数。

button_group_t是widget\_t的子类控件，widget_t的函数均适用于button_group_t控件。

在xml中使用"button\_group"标签创建button_group。如：

*<button_group x="0" y="m" w="100%" h="40" children_layout="default(c=4,r=1,s=5,m=5)">
*<button name="open:basic" text="Basic"/>
*<button name="open:button" text="Buttons"/>
*<button name="open:edit" text="Edits"/>
*<button name="open:keyboard" text="KeyBoard"/>
*</button_group>

可用通过style来设置控件的显示风格，如背景颜色等。如：

*<style name="default" border_color="#a0a0a0">
*<normal bg_color="#f0f0f0" />
*</style>
