◆ GetClosable()

window_closable_t TWindowBase::GetClosable ( ) const

收到EVT_REQUEST_CLOSE_WINDOW是否自动关闭窗口。

如果关闭窗口时，需要用户确认:

1.将closable设置为WINDOW_CLOSABLE_CONFIRM

2.处理窗口的EVT_REQUEST_CLOSE_WINDOW事件

closable在XML中取值为：yes/no/confirm，缺省为yes。
