详细描述

矢量图画布抽象基类。

具体实现时可以使用agg，nanovg, cairo和skia等方式。

cairo和skia体积太大，不适合嵌入式平台，但在PC平台也是一种选择。

目前我们只提供了基于nanovg的实现，支持软件渲染和硬件渲染。

我们对nanovg进行了一些改进:

可以用agg/agge实现软件渲染(暂时不支持文本绘制)。

可以用bgfx使用DirectX(Windows平台)和Metal(iOS)平台硬件加速。
