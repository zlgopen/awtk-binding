详细描述

可水平滚动的文本控件，方便实现长文本滚动。

hscroll_label_t是widget\_t的子类控件，widget_t的函数均适用于hscroll_label_t控件。

更多用法请参考：hscroll_label.xml

可用通过style来设置控件的显示风格，如字体的大小和颜色等等。如：

*<style name="default" text_color="black">
*<normal />
*<focused />
*</style>

更多用法请参考：

theme default
